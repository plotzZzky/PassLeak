PassLeak

Um jeito simples de verificar a segurança das suas senhas.

Nosso serviço verifica sua senha em bases de dados de vazamentos conhecidos sem armazenar ou registrar qualquer informação sensível.

Duvias frequentes:

O que é o PassLeak?

O PassLeak é uma ferramenta gratuita e segura para verificar se suas senhas foram expostas em vazamentos de dados conhecidos. Nosso serviço compara sua senha com bancos de dados públicos de brechas de segurança, sem armazenar suas informações pessoais.

Como funciona a verificação de senha?

Quando você insere sua senha no PassLeak, ela é comparada com uma base de dados de senhas vazadas. Utilizamos técnicas seguras para garantir que suas credenciais não sejam armazenadas ou expostas durante o processo. Após a verificação, você verá se sua senha foi encontrada em algum vazamento.

Minha senha será armazenada ou compartilhada?

Não. Nunca armazenamos nem compartilhamos suas senhas. A verificação ocorre de forma anônima e segura, utilizando técnicas de hashing, o que significa que seus dados não ficam registrados em nossos servidores.

Minha senha está segura após a verificação?

Após a verificação, não armazenamos nenhuma informação sua. Se sua senha estiver exposta, nossa recomendação é que você a mude imediatamente. Use autenticação de dois fatores (2FA) sempre que possível para aumentar a segurança.

Se minha senha não foi vazada, o que faço?

Não se preocupe, mas só não esqueça de mudá-la de vez em quando e usar a autenticação em dois fatores (2FA).

Se minha senha foi vazada, o que faço?

Troque por uma senha mais segura com uma mistura de letras, números e símbolos. E, para ficar ainda mais seguro, ative o 2FA.

Como posso criar uma senha segura?

- Use pelo menos 12 caracteres, incluindo números, letras (maiúsculas e minúsculas) e símbolos.

- Nunca use informações pessoais óbvias (como nome, data de nascimento, etc.).

- Evite usar palavras comuns da sua língua ou de outras.

- Considere usar um gerenciador de senhas para armazenar senhas longas e únicas para cada serviço.
